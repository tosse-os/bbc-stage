Polylang String Importer

1. Die Datei imports/translations.csv bearbeiten.
2. Das Plugin aktivieren.
3. Werkzeuge > Polylang String-Import öffnen.
4. Import starten.

CSV-Beispiel:
source;de;en
About;Über uns;About
Services;Leistungen;Services

Die Sprachspalten müssen den Polylang-Sprachkürzeln entsprechen.
Vor dem Import kann automatisch ein JSON-Backup im Upload-Verzeichnis erstellt werden.
